[게임 전제]
- 특수 블록 '폭탄팡'은 7 종류의 블록에 포함됨.
  ex. 쥐 일반 블록과 쥐 '폭탄팡' 블록은 같은 쥐 종류임.

- 특수 블록 '랜덤팡'은 7 종류의 블록에 포함되지 않음.


[1. 블록 파괴 조건]
1_1. 가로 또는 세로로 같은 종류의 블록이 3개/4개/5개 배열 시

1_2. 특수 블록 '랜덤팡' 클릭 시


[2. 파괴 조건별 효과]
2_1. 특수 블록 미포함 같은 종류 일반 블록 3개 배열 시 효과:
배열된 블록 3개 제거됨.

2_2. 특수 블록 미포함 같은 종류 일반 블록 4개/5개 배열 시 효과:
배열된 블록 4개/5개가 제거되고, 배열되었던 블록 중 사용자가 이동시킨 블록의 위치에 특수 블록이 생성됨.
4개 배열 시 특수 블록 '폭탄팡', 5개 배열 시 '랜덤팡' 생성됨.

2_3. 특수 블록 '폭탄팡' 포함 같은 종류 블록 3개 배열 시 효과:
배열된 블록 3개와 '폭탄팡' 블록 기준 상하좌우 대각선에 위치한 블록 제거됨.

(EXAMPLE)
2_3_0. '폭탄팡' 효과 범위 블록: @, 같은 종류 블록: x, 블록 x와 같은 종류의 특수 블록 '폭탄팡': X
(o 외의 다른 문자로 표시된 블록은 모두 제거됨.)

2_3_1. '폭탄팡' 블록이 같은 종류 블록 3개의 중앙에 배열됐을 때:
o o o o o o o
o o o o o o o
o o @ @ @ o o
o o x X x o o
o o @ @ @ o o
o o o o o o o
o o o o o o o

2_3_2. '폭탄팡' 블록이 같은 종류 블록 3개의 좌 또는 우에 배열됐을 때:
o o o o o o o
o o o o o o o
o o @ @ @ o o
o o @ X x x o
o o @ @ @ o o
o o o o o o o
o o o o o o o

2_3_3. '폭탄팡' 블록의 효과 범위가 blockframe 밖을 벗어날 때:
X x x o o o o
@ @ o o o o o
o o o o o @ @
o o o o x x X
o o o o o @ @
o @ @ @ o o o
o x X x o o o

2_4. 특수 블록 '폭탄팡' 포함 같은 종류 블록 4개/5개 배열 시 효과:
배열된 블록 4개/5개가 제거되고, '폭탄팡' 블록 기준 상하좌우 대각선에 위치한 블록 제거됨.
배열되었던 블록 중 사용자가 이동시킨 블록의 위치에 갯수에 따른 새로운 특수블록이 생성되며, 
새로 생성된 특수 블록은 배열되었던 '폭탄팡' 블록의 효과로는 제거되지 않음.
4개 배열 시 특수 블록 '폭탄팡', 5개 배열 시 '랜덤팡' 생성됨.

(EXAMPLE)
2_4_0. '폭탄팡' 효과 범위 블록: @, 같은 종류 블록: x, 블록 x와 같은 종류의 특수 블록 '폭탄팡': X,
사용자가 이동시킨 블록: ( ), 새로 생성되는 아무 특수 블록: N
(N을 제외하고 o 외의 다른 문자로 표시된 블록은 모두 제거됨.)

2_4_1. '폭탄팡' 블록이 같은 종류 블록 4개로 배열됐을 때:
o o o o o o o		o o o o o o o		o o o o o o o
o o o o o o o		o o o o o o o		o o o o o o o
o o o o o o o		o o @ @ @ o o		o o o o o o o
o x(x)X x o o	>>>>>	o x N X x o o	>>>>>	o o o o o o o
o o o o o o o		o o @ @ @ o o		o o N o o o o
o o o o o o o		o o o o o o o		o o o o o o o
o o o o o o o		o o o o o o o		o o o o o o o

2_5. 특수 블록 '랜덤팡' 클릭 시 효과:
blockframe에 있는 7 종류의 블록 중 랜덤한 한 종류의 블록이 갯수 상관없이 전부 제거됨.

(EXAMPLE)
2_5_0. '랜덤팡' 효과 범위 블록: #, 특수 블록 '랜덤팡': R
(@로 표시된 블록은 모두 같은 종류임.)

2_5_1. '랜덤팡' 블록 클릭 시:
o o # o o o o
o o o o o R o
# # o o # o o
o o o o o o o
o # o o o o #
o o o o o # o
# o o o o # o
